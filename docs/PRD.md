# PRD: NextPick

## 목표
1인 크리에이터가 "다음에 뭘 찍지?"를 데이터 기반으로 5분 안에 결정하도록, 내 채널 분석 + 내 니치의 트렌드 + 근거 있는 콘텐츠 추천을 한 화면에서 제공한다.

## 사용자
혼자 운영하는 YouTube 크리에이터(본인). 분석 대상 채널은 `.env`에 `YOUTUBE_CHANNEL_HANDLE`로 고정한다. 매주 주제 선정에 시간을 쓰며, 트렌드가 내 채널에 맞는지 판단이 어렵다.

## 핵심 기능
1. **내 채널 분석** — 최근 영상에서 핵심 4개 지표(평균 조회수 / 주력 키워드 / 업로드 주기 / 베스트 영상 TOP3)를 추출해 보여준다.
2. **지금 뜨는 트렌드** — 한국(KR) 인기 차트 + 내 채널 키워드 검색을 합쳐, 조회속도(virality)가 높은 영상을 내 채널과의 관련도와 함께 보여준다.
3. **다음 콘텐츠 추천** — virality × relevance 점수로 랭킹한 추천 아이디어를, 근거(어떤 트렌드·어떤 겹친 키워드)와 소스 영상 링크를 포함한 풍부한 카드로 제시한다.

## MVP 제외 사항
- LLM 추천 (휴리스틱 점수만 사용) 및 2차 API 키
- 데이터베이스 / 캐시 레이어
- 로그인 / 인증 / 다중 채널 (채널은 .env 단일 고정)
- 상태관리 라이브러리, 백그라운드 잡 / 스케줄링
- 다국가 (지역은 KR 기본값)

## 디자인
- 다크모드 고정. 매일 쓰는 분석 "도구" 톤 — 데이터 밀도 우선, 마케팅 톤 금지.
- 무채색 배경 + 상승/긍정 포인트 1색(초록). 자세한 규칙은 UI_GUIDE 참조.
